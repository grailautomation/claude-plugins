# Key Codes Reference

### Common Key Codes
- Letters: `a` through `z`
- Numbers: `1` through `0`
- F-keys: `f1` through `f20`
- Modifiers: `left_control`, `right_control`, `left_shift`, `right_shift`, `left_option`, `right_option`, `left_command`, `right_command`, `fn`
- Navigation: `up_arrow`, `down_arrow`, `left_arrow`, `right_arrow`, `page_up`, `page_down`, `home`, `end`
- Editing: `delete_or_backspace`, `delete_forward`, `return_or_enter`, `escape`, `tab`, `spacebar`
- Special: `caps_lock`, `vk_none` (no-op key)

### Consumer Key Codes (Media)
- `volume_increment`, `volume_decrement`, `mute`
- `display_brightness_increment`, `display_brightness_decrement`
- `rewind`, `play_or_pause`, `fastforward`

### Apple Vendor Key Codes
- `mission_control`, `launchpad`, `spotlight`
